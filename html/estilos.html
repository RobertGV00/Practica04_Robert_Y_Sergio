¿Qué es CSS?

Las hojas de estilo en cascada (CSS) se utilizan para dar formato al diseño de una página web.

El estilo css se puede agregar al HTML de tres maneras:

  • En la misma linea de la etiqueta HTML
  • Interno: En el apartado de head del HTML
  • Externo: En una hoja de estilos aparte.
En linea

Se utiliza un CSS en línea para aplicar un estilo único a un único elemento HTML. Pondre un ejemplo en el siguiente parrafo haciendo el texto de otro color

Prueba de estilo en linea

Interno

Un css interno se define en el head de la pagina y se guarda dentro de una seccion llamada style.

Externo

El css externo es el que se realiza en una hoja de estilos aparte. Es el que estamos usando durante esta practica.

Para ver ejemplos de todo lo que podemos hacer y cuanto podemos afinar con esto, lo podemos ver en la hoja de estilos que estamos usando para la practica.

Para vincular el HTML con la hoja de estilos, se hace mediante la etiqueta link

Algunas propiedades CSS

Para dar color a un texto se hace con la propiedad 'color'

Si quieres definir una fuente para usar que no sea la por defecto, se hace con 'font-family'

Para definir el tamaño del texto, usaremos 'font-size'

Para definir un borde alrededor de un elemento HTML se hace con la etiqueta 'border'

Para definir un espacio entre el texto y el borde, se hace con la propiedad 'padding'

Para definir un margen fuera del borde, se hace con la propiedad 'margin'
