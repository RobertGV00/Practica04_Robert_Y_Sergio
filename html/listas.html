Listas Desordenadas

Una lista desordenada, empieza con un <ul> y acaba con un </ul>.
Y luego usamos un <li> para escribir dentro de ellas. </li>

Listas Ordenadas

Una lista ordenadas, empieza con un <ol> y acaba con un </ol>.
Y luego usamos un <li> para escribir dentro de ellas. </li>

Ordenada con numeros decimales

  1. Articulo 1
  2. Articulo 2
  3. Articulo 3
  4. Articulo 4

Ordenada con numeros romanos a difernte nivel

  1. Articulo 1
    1. Artiulo 1.1
    2. Articulo 1.2

Listas Descriptivas

Una lista descriptiva es una lista de terminos con una descripción de cada uno.
Para usar esta lista, lo primero que debemos hacer es abrir la etiqueta <dl> que define la lista, y dentro de esta etiqueta usamos primero la etiqueta <dt> que define los terminos, y la ultima que usariamos sería la etiqueta <dd> que sería la descripción del objeto. todas estas etiquetas se deben cerrar al acabar con ellas.
