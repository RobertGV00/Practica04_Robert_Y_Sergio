Los enlaces permiten a los usuarios hacer clic para pasar de una página a otra.

HIPERVÍNCULOS

Los enlaces HTML son hipervínculos.

Puede hacer clic en un enlace y saltar a otro documento.

Cuando mueves el mouse sobre un enlace, la flecha del mouse se convertirá en una manita.

ENLACES HTML: Sintaxis

La etiqueta HTML a define un hipervínculo. Tiene la siguiente sintaxis:

< a href="url" link text >;

El atributo más importante del < a > elemento es el hrefatributo, que indica el destino del enlace.

El texto del enlace es la parte que será visible para el lector.

Al hacer clic en el texto del enlace, se enviará al lector a la dirección URL especificada.

Ejemplo

Este ejemplo muestra cómo crear un enlace a W3Schools.com:

< a href= "https://www.W3Schools.com/Visit W3Schools.com! > a

Nota: Los enlaces se pueden diseñar con CSS para darle otro aspecto

ENLACES HTML: El atributo de destino

la página vinculada se mostrará en la ventana actual del navegador. Para cambiar esto, debe especificar otro destino para el enlace.

El atributo target especifíca dónde abrir el documento vinculado

Puede tener uno de los siguientes valores:

  • _self: Por defecto abre el documento en la misma ventana en la que se hizo clic.
  • _blank: Abre el documento en una nueva ventana.
  • _parent: Abre el documento en el marco principal.
  • _top: Abre el documento en el cuerpo completo de la ventana.

mailto: dentro del href para crear un enlace que abra el programa de correo electrónico del ususario ( para permitirle enviar un nuevo correo electrónico).

Ejemplo

Títulos de enlaces

El atributo title especifíca información adicional sobre un elemento.Suele mostrarse como texto de información sobre herramientas cuando el ratón se mueve sobre el elemento.

Ejemplo

<a href = "https://wwww____/ title = "___" >___</a>

ENLACES HTML: Crear marcadores

Pueden resultar útiles si la página es muy larga.

Para crear un marcador, primero cree el marcador y luego agréguele un enlace.Cuando se hace clic en el enlace, la página se desplazará hacia arriba o hacia abajo hasta la ubicación con el marcador.

Ejemplo

Use id para crear un marcador
<h2 id = "C4">Capítulo 4</h2>
Luego agréguele un enlace al marcador(saltar al capítulo 4) desde la misma página.
<a href= "#C4">saltar capítulo 4</a>
También puedes agregar un enlace a un marcador en otra página
