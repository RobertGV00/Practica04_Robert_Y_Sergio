Etiquetas Basicas

Parrafo
Si quieres especificar que se trata de un párrafo, podrías encerrar el texto con la etiqueta de párrafo <p> Entre esta etiqueta va el parrafo deseado.</p>
Anidar Elementos

Puedes también colocar elementos dentro de otros elementos —esto se llama anidamiento

Como ejemplo usaremos la etiqueta <strong> que lo que hace es enfatizar una palabra o frase.

<Texto enfatizado> Texto sin enfatizar

Muy importante, si una etiqueta la abres dentro de la <p>, cierrala dentro de la </p>, no fuera de ella.

Elementos Vacios

Algunos elementos no poseen contenido, y son llamados elementos vacíos. Por ejemplo <img>. Sobre la etiqueta <img> tenemos mas información en su apartado correspondiente.

Marcado de Texto

Esta sección cubrirá algunos de los elementos HTML básicos que usarás para el marcado de texto.

Encabezados

Los elementos de encabezado permiten especificar que ciertas partes del contenido son encabezados, o subencabezados del contenido

Los encabezados poseen seis niveles, desde <h1> hasta <h6>

Listas

Mucho del contenido web está dado por listas, así que HTML tiene elementos especiales para ellas. Veremos mas sobre ellas en su apartado respectivo.

Vinculos

Los vínculos o enlaces son muy importantes, son los que hacen de la web, la web. Veremos mas sobre ellos en su respectivo apartado.

Anatomía de un documento HTML

Hasta ahora has visto lo básico de elementos HTML individuales, pero estos no son muy útiles por sí solos. Ahora verás cómo los elementos individuales son combinados para formar una página HTML entera

<!DOCTYPE html>

El tipo de documento. Es un preámbulo requerido

<html></html>

Este elemento encierra todo el contenido de la página entera y, a veces, se le conoce como el elemento raíz

<head></head>

Este elemento actúa como un contenedor de todo aquello que quieres incluir en la página HTML que no es contenido visible por los visitantes de la página

<meta charset="utf-8">

Este elemento establece el juego de caracteres que tu documento usará en utf-8, que incluye casi todos los caracteres de todos los idiomas humanos

<title></title>

Este elemento establece el título de tu página, que es el título que aparece en la pestaña o en la barra de título del navegador cuando la página es cargada

<body></body>

Encierra todo el contenido que deseas mostrar a los usuarios web que visiten tu página, ya sea texto, imágenes, videos, juegos, pistas de audio reproducibles, y demás.
