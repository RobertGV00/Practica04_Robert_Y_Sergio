Elementos

Los elementos de formato se diseñaron para mostrar tipos especiales de texto:

Etiqueta <b>

Define texto en negrita como esto: Prueba de texto en negrita

Etiqueta <strong>

Define el texto con gran importancia. El contenido del interior normalmente se muestra en negrita. Es como el b, pero este suele tener mas importancia. Prueba de texto con importancia

Etiqueta <i>

El contenido interior normalmente se muestra en cursiva. Prueba de cursiva

Etiqueta <em>

Define el texto enfatizado. El contenido interior normalmente se muestra en cursiva. Prueba de texto enfatizado

Etiqueta <mark>

Define el texto que debe marcarse o resaltarse. Resaltado

Etiqueta <small>

Define un texto que es mas pequeño. Texto pequeño

Etiqueta <del>

Esta etiqueta define un elemento 'eliminado', marca un texto como tachado. Texto tachado

Etiqueta <ins>

Es un texto subrayado. Texto subrayado

Etiqueta <sub>

Define un texto de subindice. H2O

Etiqueta <sup>

Define un texto de superindice. Esto es unaprueba de superindice
