¿Que es HTML?

HTML, o HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es un estándar utilizado para crear y diseñar páginas web. Define la estructura y organización del contenido mediante etiquetas y atributos, permitiendo la creación de enlaces, imágenes, formularios y otros elementos interactivos en la web.
